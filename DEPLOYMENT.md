# Deploy Eman Aijaz Learning Hub

This is a static website. Upload these files together:

- `index.html`
- `styles.css`
- `script.js`
- `assets/learning-hero.png`

Fast Netlify option:

1. Open https://app.netlify.com/drop
2. Drag the `outputs` folder into the page.
3. Wait for Netlify to publish the site.
4. Copy the public website link Netlify gives you.

GitHub Pages option:

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and the `assets` folder.
3. Go to repository Settings > Pages.
4. Set source to the main branch and root folder.
5. Save and wait for the public website link.
